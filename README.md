# webscraping_project2
🌐 Web Scraping Project – Largest U.S. Companies by Revenue

In this project, I performed web scraping to extract and analyze data from a public website.

🔧 Tools & Libraries
Python
BeautifulSoup
Requests
Pandas
📌 Project Steps
Imported the required libraries: BeautifulSoup and requests.
Identified and accessed the target URL using Google search.
Located the table containing the list of the largest companies in the United States by revenue within the HTML code.
Cleaned the data by removing unnecessary titles and elements.
Created structured table headers using Pandas.
Extracted the relevant data from the website.
Organized the data into a DataFrame.
Exported the final dataset into a .csv file for further analysis.
✅ Outcome

The project demonstrates how to:

Extract structured data from a website
Clean and organize raw HTML data
Store data in a usable format (CSV
